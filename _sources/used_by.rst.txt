
=======
Used By
=======

Projects using ``asv-perch`` for benchmark CI.

`eOn <https://github.com/TheochemUI/eOn>`_
    Saddle point search methods
    for atomic-scale simulations. Uses the full pipeline pattern with pixi,
    meson-built C++ libraries, and ``comparison-text-file`` mode.
    (`benchmark workflow <https://github.com/TheochemUI/eOn/blob/main/.github/workflows/ci_benchmark.yml>`_,
    `commenter workflow <https://github.com/TheochemUI/eOn/blob/main/.github/workflows/ci_bench_commenter.yml>`_)

`rgpycrumbs <https://github.com/HaoZeke/rgpycrumbs>`_
    Dispatcher-based
    analytical suite for chemical physics. Benchmarks JAX kernel regression,
    scipy B-spline interpolation, and ASE structure analysis via pixi environments
    and ``comparison-text-file`` mode.
    (`benchmark workflow <https://github.com/HaoZeke/rgpycrumbs/blob/main/.github/workflows/ci_benchmark.yml>`_,
    `commenter workflow <https://github.com/HaoZeke/rgpycrumbs/blob/main/.github/workflows/ci_bench_commenter.yml>`_)
